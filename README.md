# MingYao Salsabila

A small private app made with love for my girlfriend.

This project is a personal dashboard for daily care, routines, reminders, notes, and gentle tracking.  
It is designed to feel warm, simple, and local-first.

## Purpose

This app exists to help me share:
- care
- consistency
- attention
- small daily reminders
- private moments we can keep in one place

## Features

- Local-first storage
- Offline use
- Daily routines
- Notes and journal
- Cycle and mood tracking
- Skincare / self-care logging
- Google Calendar sync support
- Time-based reminders
- Mobile-friendly layout

## Privacy

This project is meant to stay personal and private.

- Data is stored locally first
- No public sharing of sensitive data
- Sync is optional
- Designed for personal use only

## Tech Stack

- HTML
- CSS
- JavaScript
- IndexedDB
- PWA support
- Google Calendar API

## Roadmap

- Improve sync stability
- Add local backup and restore
- Add better offline support
- Add cleaner mobile UX
- Add more personal care features

## Deployment

This project can be deployed as a static web app and installed on mobile as a PWA.

## Note

This is not a commercial product.  
It is a small project made to show love, care, and intention.
